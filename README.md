## Usage

```bash
# compilation
cd qrhi-tutorials && mkdir build && cd build

cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
```

### Environment

- `C++` :  `C++20`
- `Qt` : `6.8.x`
- `CMake` : `>= 3.22`
